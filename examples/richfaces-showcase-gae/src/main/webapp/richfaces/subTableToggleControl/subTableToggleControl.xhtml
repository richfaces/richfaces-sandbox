This sample shows simple master-detail table implemented using rich:dataTable and rich:subtable components

There is one feature which is completely new for 4.x - subtables now could be collapsed/expanded by using new component rich:tableToggleControl

Switching customization:

  • subTable's could be collapsed/expanded in different modes defined via subTable expandMode attribute - "client", "ajax" and "server"
  • The developer could get control on current states of subTables using its expanded attribute.
